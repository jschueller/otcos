# otcos module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otcos\examples\otcos

Installation
  Silent install can be done with the switch /S (e.g. openturns-otcos-0.0-py27.exe /S).
  To test otcos module installation, in an OpenTURNS python console type:
    import otcos
    print otcos.__version__

Uninstallation
  otcos module will be automatically uninstalled at OpenTURNS removal.
  otcos module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
